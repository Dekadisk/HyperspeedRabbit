	LANCEMENT

	Ouvrir en Release/Debug x64
/////////////////////////////////////////////////////////////////////////////
	COMMANDES

ZQSD(ouWASDselon la langue du clavier) : bouger le vaisseau dans le plan
Souris: viser
Clic gauche: tirer
SPACEBAR: accélérer
Left SHIFT: ralentir
TAB: cycler dans les caméras
ECHAP: mettre le jeu en pause, et afficher le menu de pause
R: redemarrer le jeu
/////////////////////////////////////////////////////////////////////////////
Pour plus d'informations consulter la  page accéssible depuis le jeu en appuyant sur le bouton "Help" dans le menu du jeu
